🍪 Cookie 设置

设置 Cookie 后可获取高清原图和视频。Cookie 仅存储在本地服务器，不会上传到其他位置。 如何获取 Cookie？

如何获取 Cookie：

方法 1：使用浏览器扩展（推荐）
  1. 安装 Cookie 导出扩展（如 "Get cookies.txt" 或 "Cookie-Editor"）
  2. 在浏览器中打开 小红书 或 抖音 网站并登录
  3. 点击扩展图标，选择导出/复制 Cookie
  4. 直接粘贴到上面的输入框（支持 Netscape 格式自动转换）
方法 2：手动从开发者工具复制
  1. 在浏览器中打开 小红书 或 抖音 网站并登录
  2. 按 F12 打开开发者工具，切换到 Network（网络）标签
  3. 刷新页面，点击任意一个请求
  4. 在右侧找到 Headers（请求头），复制 Cookie 字段的值

提示：支持两种格式：1) name=value; name2=value2 2) Netscape HTTP Cookie File 格式
